# Site Configuration

This file contains the site configuration including:

- **site_name**: The name displayed in the header
- **tagline**: The subtitle displayed next to the site name
- **window_title**: The title shown in the browser tab (optional, defaults to site_name)

Configure these values to customize your documentation site.
